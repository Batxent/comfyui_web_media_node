# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Focus feedback on code and behavior, not people.
- Assume good intent and ask clarifying questions first.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Trolling, insulting comments, or public shaming.
- Publishing private information without permission.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, and issues that violate this policy.
